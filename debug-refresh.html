Common Causes

1. JavaScript Redirect Loop

Check if there's a redirect in your JavaScript files:

  • Look for window.location.href or window.location.replace
  • Check for automatic redirects in main.js
  • Look for admin authentication redirects

2. PHP Redirect Issues

If you have PHP files, check for:

  • header("Location: ...") redirects
  • Session redirects
  • Installation script redirects

3. Meta Refresh Tag

Check your HTML for meta refresh tags:

<meta http-equiv="refresh" content="0;url=...">

4. Browser Cache Issues

Try these solutions:

  • Clear browser cache (Ctrl+Shift+Delete)
  • Try incognito/private browsing mode
  • Try a different browser

Quick Fixes

Fix 1: Check main.js

Open assets/js/main.js and look for any redirect code. Comment out suspicious lines:

// window.location.href = 'some-page.html';

Fix 2: Check admin.js

Open admin/admin.js and look for authentication redirects:

// this.redirectToDashboard();

Fix 3: Remove install.php

If you have install.php, rename or delete it after installation:

mv install.php install.php.bak

Fix 4: Check .htaccess

Look for any .htaccess files that might have redirects:

RewriteRule ^(.*)$ index.html [L]

Emergency Fix

If nothing else works, try this temporary fix:

  1. Rename assets/js/main.js to assets/js/main.js.bak
  2. Rename admin/admin.js to admin/admin.js.bak
  3. Refresh the page
  4. If it works, the issue is in one of these JavaScript files
