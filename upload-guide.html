File Upload Guide

Welcome! This guide will help you upload your logo and product catalogues to the website.

Logo Files

Upload your logo files to the assets/images/ directory:

  • logo.png - Main logo (recommended: 200x40px)
  • logo-white.png - White version for dark backgrounds
  • logo-small.png - Smaller version for mobile (150x30px)
  • favicon.ico - Website favicon (16x16px or 32x32px)
Logo Requirements:
  • Use PNG format for best quality
  • Keep file size under 100KB
  • Use transparent background if possible
  • Test on both light and dark backgrounds

Product Catalogues

Upload your product catalogues to the assets/documents/ directory:

  • product-catalogue.pdf - Complete product catalogue
  • security-catalogue.pdf - Security cameras catalogue
  • automation-catalogue.pdf - Home automation catalogue
  • boom-barrier-catalogue.pdf - Boom barriers catalogue
  • brochure.pdf - Company brochure
  • price-list.pdf - Current price list
PDF Requirements:
  • Keep file size under 10MB for fast loading
  • Optimize images within PDFs
  • Use clear, readable fonts
  • Include contact information in catalogues

Upload Steps

  1. Access your website files - Use FTP, file manager, or your hosting control panel
  2. Navigate to the correct directories:
    • Logo files: ambicamarketing/assets/images/
    • Catalogues: ambicamarketing/assets/documents/
  3. Upload your files with the exact names listed above
  4. Test the website to ensure everything displays correctly
