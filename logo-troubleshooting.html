Logo Troubleshooting Guide

Let's fix your logo display issue! Follow these steps to get your logo showing properly.

Step 1: Check File Location

Your logo should be located at:

ambicamarketing/assets/images/logo.png

Common issues:

  • File is in wrong folder
  • File name is different (case-sensitive)
  • File extension is wrong (.jpg instead of .png)

Step 2: Check File Name

Your logo file must be named exactly:

logo.png

NOT:

  • Logo.png (capital L)
  • logo.PNG (capital extension)
  • logo.jpg or logo.jpeg
  • ambica-logo.png
  • company-logo.png

Step 3: Test Logo Display

Testing your logo:

Logo not found! Check file location and name.

Logo found and displaying!

Step 4: Common Solutions

Solution 1: Rename Your File

If your logo has a different name, rename it to logo.png

Solution 2: Convert File Format

If your logo is JPG, convert it to PNG format

Solution 3: Check File Path

Make sure the file is in the correct folder: assets/images/

Solution 4: Clear Browser Cache

Press Ctrl+F5 (or Cmd+Shift+R on Mac) to refresh without cache

Step 5: Alternative File Names

If you want to use a different filename, update the HTML code:

<img src="assets/images/YOUR-LOGO-FILENAME.png" alt="Ambica Marketing" class="logo-img">

Common alternatives:

  • assets/images/ambica-logo.png
  • assets/images/company-logo.png
  • assets/images/logo.jpg (if using JPG)

Step 6: Verify Success

After making changes:

  1. Save your file with the correct name
  2. Clear browser cache (Ctrl+F5)
  3. Refresh the page
  4. Check if logo appears in the header

Still Not Working?

If your logo still doesn't appear:

  • Check file permissions (should be readable)
  • Verify file isn't corrupted
  • Try a different browser
  • Check browser developer tools (F12) for errors
